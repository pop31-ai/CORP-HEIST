# CORP HEIST

Корпоративный idle/hero-battler на Python + Pygame.

## Запуск

```
pip install pygame
python main.py
```

## Управление

- **ЛКМ** — выбор / действие
- **Esc** — назад
- **S** — сохранение
- **Колесо мыши** — прокрутка

## Содержание

- 10 героев (B → SSR тиры)
- 5 корпораций с бонусами
- 7 этажей бизнес-центров
- Дерево пассивных навыков
- Гача с pity-системой
- Боевая система (ход за ходом)
- Магазин + премиум-донат
- JSON-сохранения
- Предупреждения и 14-ФЗ compliance
